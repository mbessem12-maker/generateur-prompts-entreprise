
# Générateur de Prompts pour Mon Entreprise

Ce projet permet de générer des prompts personnalisés pour créer du contenu (textes, images) adapté aux besoins de mon entreprise.

## Fonctionnalités
- Génération de prompts pour des articles, posts, images, etc.
- Base de données de termes réutilisables.
- Exemples prêts à l'emploi.

## Utilisation
1. Clone ce dépôt.
2. Exécute `python generateur_prompts.py`.
3. Personnalise les prompts selon tes besoins.
